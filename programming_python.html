Comments

In python comments are 2 types.
They are

  • Single line comment: These are start with #
    Example:
    # This is single line comment
  • Multi-line comment: These are start and end with ''' or """
    Example:
    '''This is multi-line comment
    Now you can write your comments
    in multiple lines'''

    """This is multi-line comment
    Now you can write your comments
    in multiple lines"""
